# 合约交易使用指南

## 🚀 **新增功能**

我已经为你在 `main.py` 中添加了完整的合约交易功能，包括：

### 1. **合约交易方法**
- `execute_futures_trade()` - 执行合约交易
- `get_futures_position()` - 获取合约持仓信息
- `calculate_futures_position_size()` - 计算合约开仓数量
- `close_futures_position()` - 平仓合约持仓

### 2. **合约交易主程序**
- `futures_trading_main()` - 完整的合约交易主循环
- 支持杠杆交易（默认10倍杠杆）
- 自动止盈止损
- 持仓监控和风险管理

### 3. **智能启动脚本**
- `run_trading.py` - 支持选择现货或合约交易模式

## 📊 **合约交易参数**

| 参数 | 值 | 说明 |
|------|-----|------|
| 交易对 | BTC-USDT-SWAP | 永续合约 |
| 杠杆倍数 | 10x | 可调整 |
| 保证金模式 | 全仓 | 使用全部可用保证金 |
| 风险比例 | 2% | 单次开仓风险控制 |
| 止盈设置 | 1.0% | 策略止盈 |
| 止损设置 | 0.5% | 策略止损 |

## 🔧 **使用方法**

### **方法1: 使用启动脚本（推荐）**

## ⚡ **合约交易特点**

### **优势**
1. **杠杆效应**: 10倍杠杆，小资金大收益
2. **双向交易**: 可以做多做空
3. **资金效率**: 保证金交易，资金利用率高
4. **风险控制**: 自动止盈止损，强制平仓保护

### **风险提示**
1. **杠杆风险**: 10倍杠杆意味着10倍风险
2. **强制平仓**: 保证金不足时会被强制平仓
3. **市场风险**: 合约价格波动可能很大
4. **时间价值**: 永续合约有资金费率

## 📈 **交易逻辑**

### **开仓条件**
1. 策略生成买入信号
2. 当前无持仓
3. 今日未交易
4. 风控检查通过

### **平仓条件**
1. 策略生成卖出信号
2. 止盈触发（盈利1.0%）
3. 止损触发（亏损0.5%）
4. 手动平仓

### **仓位管理**
- 基于可用资金和杠杆计算开仓数量
- 风险比例控制在2%
- 全仓模式，最大化资金利用
