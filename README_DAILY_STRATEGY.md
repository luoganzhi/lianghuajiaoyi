# 每日交易策略使用指南

## 🎯 策略概述

这是一个基于RSI、MA和成交量指标的每日交易策略，具有以下特点：

- **每日交易限制**：每个交易日只交易一次
- **高胜率**：回测显示55.6%的胜率
- **风险可控**：1.0%止盈，0.5%止损
- **跨日持仓**：支持跨日持仓，等待合适时机卖出

## 🚀 快速启动

### 1. 环境准备

确保已安装所需依赖：
```bash
pip install -r requirements.txt
```

### 2. 配置检查

检查 `src/config/config.py` 中的配置：
- `IS_SIMULATED = True` - 使用模拟盘
- API密钥配置正确
- 代理设置正确（如需要）

### 3. 启动策略

```bash
# 方法1：使用启动脚本
python run_daily_strategy.py

# 方法2：直接运行main.py
PYTHONPATH=/path/to/project python src/main.py
```

## 📊 策略参数

### 核心参数
- **止盈**：1.0%
- **止损**：0.5%
- **RSI周期**：21
- **MA设置**：短期5，长期20
- **交易时间**：0:00-24:00（全天）

### 买入条件
1. RSI在35-65区间且反弹
2. 价格在长期MA之上
3. 价格回调0.5%-3%
4. 成交量放大1.1倍以上

## 🔧 策略特点

### ✅ 优势
- **高胜率**：55.6%的胜率
- **风险可控**：最大回撤1.86%
- **每日限制**：避免过度交易
- **跨日持仓**：灵活处理持仓

### ⚠️ 注意事项
- 每日只能交易一次
- 需要稳定的网络连接
- 建议在模拟盘充分测试后再实盘

## 📈 监控指标

策略运行时会显示：
- 当前价格和账户余额
- 策略持仓状态
- 实时盈亏情况
- 交易记录和统计

## 🛠️ 自定义配置

### 修改策略参数
在 `src/main.py` 中修改策略初始化参数：

```python
strategy = DailyTradingStrategy(
    take_profit=0.01,      # 修改止盈比例
    stop_loss=0.005,       # 修改止损比例
    rsi_period=21,         # 修改RSI周期
    # ... 其他参数
)
```

### 启用优化功能
可以启用第一阶段的优化功能：

```python
strategy = DailyTradingStrategy(
    # ... 基础参数
    use_dynamic_stop=True,  # 启用动态止损
    use_macd=True,         # 启用MACD确认
    avoid_hours=[2, 3, 4], # 避开高波动时段
    best_hours=[9, 10, 14, 15, 20, 21], # 最佳交易时段
)
```
